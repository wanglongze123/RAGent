## 1. 对话接口

### 1.1 发起对话（流式）

客户端发送用户消息，后端以 SSE 流式返回 AI 回复。

```
POST /api/v1/chat/stream
Content-Type: application/json
Accept: text/event-stream
```

**请求体**

```json
{
  "session_id": "sess_abc123",
  "message": "推荐一款适合油皮的洗面奶",
  "image_base64": null
}
```

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| session_id | string | 是 | 会话 ID，同一会话保持不变（多轮上下文靠此关联） |
| message | string | 是 | 用户文字输入，图片搜索时传空字符串 `""` |
| image_base64 | string\|null | 否 | 拍照找货时传 Base64 编码的图片（JPEG，建议压缩到 < 500KB） |

**响应**

`Content-Type: text/event-stream`，响应体为 SSE 事件流，见[第 6 节](#6-sse-事件流协议)。

**Android 示例**

```kotlin
val request = Request.Builder()
    .url("http://localhost:8000/api/v1/chat/stream")
    .post(body)
    .header("Accept", "text/event-stream")
    .build()

EventSources.createFactory(okHttpClient)
    .newEventSource(request, eventSourceListener)
```

### 1.3 拍照找货（多模态）

用户上传图片，后端用 `Doubao-embedding-vision` 将图片转为向量，在商品图向量库中检索相似商品，以 SSE 流式返回结果。**响应格式与 1.1 完全一致**，客户端复用同一套 SSE 解析逻辑。

```
POST /api/v1/search/by-image
Content-Type: application/json
Accept: text/event-stream
```

**请求体**

```json
{
  "session_id": "sess_abc123",
  "image_base64": "/9j/4AAQSkZJRgAB...",
  "image_mime_type": "image/jpeg"
}
```

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| session_id | string | 是 | 会话 ID，搜索结果会记入对话历史 |
| image_base64 | string | 是 | Base64 编码的图片，客户端压缩到 < 500KB 再上传 |
| image_mime_type | string | 是 | 固定传 `image/jpeg` |

**响应**

SSE 事件流，典型推送顺序：

```
event: thinking
data: {"message": "正在识别图片中的商品..."}

event: tool_progress
data: {"tool": "image_search", "message": "正在为您找相似款式..."}

event: product_card_list
data: {"products": [...], "search_type": "image"}

event: text_delta
data: {"text": "为您找到以下相似商品，您可以进一步告诉我颜色或尺码偏好。"}

event: done
data: {"session_id": "sess_abc123", "agent_state": "browsing"}
```

`product_card_list` 中每个商品额外携带 `similarity_score`（0-1 相似度），供客户端展示"匹配度"标签：

```json
{
  "products": [
    {
      "product_id": "p_apparel_003",
      "title": "宽松休闲外套春秋款",
      "image_url": "/static/images/p_apparel_003_live.jpg",
      "price": 299.0,
      "reason": "版型与上传图片相似，均为宽松直筒廓形",
      "similarity_score": 0.92
    }
  ],
  "search_type": "image"
}
```

**Android 实现要点**

```kotlin
// 1. 拍照或选图后压缩
val compressed = BitmapFactory.decodeFile(path)
    .let { Bitmap.createScaledBitmap(it, 800, 800, true) }
val base64 = Base64.encodeToString(
    ByteArrayOutputStream().also { compressed.compress(JPEG, 80, it) }.toByteArray(),
    Base64.NO_WRAP
)

// 2. 构造请求，复用与 chat/stream 相同的 SSE 监听器
val body = """{"session_id":"$sessionId","image_base64":"$base64","image_mime_type":"image/jpeg"}"""
```


**响应**

```json
{
  "cart_item_id": "ci_001",
  "message": "已加入购物车",
  "cart_total_count": 1
}
```

## 6. SSE 事件流协议

**这是客户端最重要的部分**，所有 AI 回复通过此协议推送。

### 格式规范

每条事件两行：

```
event: {事件类型}
data: {JSON字符串}

```

注意：每条事件后有一个空行作为分隔符。

---

### 事件类型总览

| 事件类型 | 触发时机 | 客户端动作 |
|---|---|---|
| `thinking` | Agent 开始处理，尚未检索 | 显示"正在思考..."动效 |
| `tool_progress` | 工具调用进行中 | 显示"正在检索商品..." |
| `text_delta` | 模型逐字生成文本 | 追加到当前气泡 |
| `product_card` | 检索到商品，推送单张卡片 | 在对话流插入商品卡片 |
| `product_card_list` | 多商品列表（条件筛选结果） | 渲染横滑商品列表 |
| `comparison_table` | 多商品对比结果 | 渲染对比表格组件 |
| `cart_update` | 购物车状态变化（对话式加购/删除） | 更新购物车角标和状态 |
| `clarification` | Agent 主动反问用户 | 显示反问气泡+可选项按钮 |
| `image_searching` | 拍照找货专用，图片向量化完成开始检索 | 更新进度提示文案 |
| `error` | 后端异常 | 显示错误提示 |
| `done` | 本轮回复结束 | 停止加载动效，允许再次输入 |

---

### 各事件 data 结构

#### `thinking`
```json
{
  "message": "正在理解您的需求..."
}
```

#### `tool_progress`
```json
{
  "tool": "hybrid_search",
  "message": "正在为您检索相关商品..."
}
```

#### `text_delta`
```json
{
  "text": "为您推荐"
}
```
客户端收到后直接拼接到当前文本气泡末尾。

#### `product_card`
```json
{
  "product_id": "p_beauty_007",
  "title": "科颜氏白泥净透洁面皂142g",
  "brand": "科颜氏",
  "image_url": "/static/images/p_beauty_007_live.jpg",
  "price": 185.0,
  "sub_category": "洁面",
  "reason": "含高岭土成分，深层清洁毛孔，适合油皮控油"
}
```

注意：`reason` 是 AI 生成的推荐理由，`price`/`title`/`image_url` 全部来自原始数据，**不由 AI 生成**。

#### `product_card_list`
```json
{
  "products": [
    {
      "product_id": "p_beauty_007",
      "title": "科颜氏白泥净透洁面皂142g",

      "image_url": "/static/images/p_beauty_012_live.jpg",
      "price": 128.0,

```


#### `cart_update`
```json
{
  "action": "add",
  "product_id": "p_beauty_007",
  "sku_id": "s_p_beauty_007_1",
  "title": "科颜氏白泥净透洁面皂142g",
  "quantity": 1,
  "cart_total_count": 2,
  "cart_total_price": 305.0,
  "message": "已将科颜氏白泥加入购物车 🛒"
}
```

`action` 枚举值：`add` / `remove` / `update_quantity`

#### `clarification`
```json
{
  "question": "请问您的预算范围是？",
  "options": ["200元以内", "200-500元", "500元以上", "不限预算"]
}
```

客户端渲染为带选项按钮的反问气泡，用户点击某个选项等同于发送该文字。

#### `error`
```json
{
  "code": "RETRIEVAL_FAILED",
  "message": "商品检索失败，请重试"
}
```

#### `done`
```json
{
  "session_id": "sess_abc123",
  "agent_state": "browsing"
}
```

`agent_state` 枚举值：`browsing` / `comparing` / `cart_management` / `checkout`

客户端可根据此字段决定底部操作栏的状态（如 checkout 状态显示"查看订单"按钮）。

## 8. 错误码规范

所有非 2xx 响应统一格式：

```json
{
  "code": "ERROR_CODE",
  "message": "人类可读的错误描述",
  "detail": "可选的调试信息"
}
```

| HTTP 状态码 | code | 含义 |
|---|---|---|
| 400 | `INVALID_PARAMS` | 请求参数错误 |
| 404 | `PRODUCT_NOT_FOUND` | 商品不存在 |
| 404 | `SESSION_NOT_FOUND` | 会话不存在 |
| 404 | `CART_ITEM_NOT_FOUND` | 购物车商品不存在 |
| 409 | `OUT_OF_STOCK` | 商品库存不足 |
| 429 | `RATE_LIMITED` | 请求过于频繁 |
| 500 | `LLM_ERROR` | 大模型调用失败 |
| 500 | `RETRIEVAL_FAILED` | 向量检索失败 |
| 503 | `SERVICE_DEGRADED` | 服务降级，返回兜底回复 |

### Mock 数据建议（给客户端同学）

后端未就绪前，客户端可用以下 mock SSE 数据本地调试：

```kotlin
// MockSseRepository.kt
val mockEvents = listOf(
    SseEvent("thinking", """{"message": "正在为您检索..."}"""),
    SseEvent("text_delta", """{"text": "为您推荐以下商品："}"""),
    SseEvent("product_card", """{"product_id":"p_beauty_007","title":"科颜氏白泥洁面皂","price":185.0,"image_url":"","reason":"适合油皮"}"""),
    SseEvent("done", """{"session_id":"mock_session","agent_state":"browsing"}""")

```

### Android 注意事项

1. **模拟器访问本机**：`localhost` 改为 `10.0.2.2`，真机调试用电脑局域网 IP
2. **HTTP 明文**：开发阶段在 `network_security_config.xml` 允许 HTTP，上线前换 HTTPS
3. **图片加载**：用 Coil 加载 `http://10.0.2.2:8000/static/images/xxx.jpg`
4. **SSE 库**：`implementation "com.squareup.okhttp3:okhttp-sse:4.12.0"`
